# 기본 디자인 토큰 — 정부 발표 스타일 (design.md 미제공 시 기본값)

> `pptx-design-analyst`가 사용자 제공 디자인 자산을 분석해 `design-ref.md`를 주거나,
> 사용자가 `design.md`를 주면 그것을 우선한다. **둘 다 없을 때 이 기본값으로 진행**한다.
> 값은 `lib/build_helpers.mjs`의 `C` 토큰과 동기화되어 있다.

## 캔버스
- 16:9, 13.333 × 7.5 inch, 좌우 마진 `MX = 0.9`, 레이아웃명 `W`.
- 폰트: **맑은 고딕**(정부표준·Pretendard 미설치 대체). Pretendard 설치 시 `setFont("Pretendard")`.

## 색 팔레트 (HEX)
| 토큰 | 값 | 용도 |
|------|----|------|
| primary | `7C6BFF` | 강조·CTA·번호배지·키워드 강조(밝은 배경) |
| primaryDk | `5A49C9` | 보라 텍스트, 부처·기관 라벨 |
| primaryLt | `EDEAFB` | 아이콘 원·옅은 바 |
| warm | `FCEFE6` | 표지·디바이더 외 본문 배경, 카드 배경 |
| lilac | `EEEAF7` | 코너 데코 원 |
| white | `FFFFFF` | 본문 슬라이드 배경, 카드 |
| dark | `241C45` | 다크(마지막 액션) 배경 |
| darkCard | `33295E` | 다크 카드 |
| ink/body/gray | `1F1B2E`/`4A4658`/`8A8696` | 본문/설명/캡션 |
| gold | `F4B740` | 키워드 강조(보라·다크 배경) |

> 새 주제로 재활용 시 `primary` 계열만 주제색으로 교체, warm/lilac은 유지하면 통일감.

## 타이포 위계 (L3 — 크게)
| 역할 | 크기 | 비고 |
|------|-----|------|
| 표지 헤드라인 | 47~50 | 강조어 primary |
| 슬라이드 제목 | 34 | `header()` |
| 디바이더 제목 | 40~44 | 보라 배경, 흰색 |
| 카드 제목 | 18~24 | |
| 본문 | 14~16 | lineSpacing 1.3~1.4 |
| eyebrow / 캡션 | 12.5 / 9.5~11 | eyebrow 대문자+charSpacing 3 |

## 형태·여백 (L5 균등)
- 카드 라운드 `rectRadius 0.14~0.18`, 보라 소프트섀도우 `shadow("9B86FF",0.28,16,5)`.
- 카드 내부 `pad(x,y,w,h,0.34)`로 좌우 동일·세로 중앙. 텍스트 `margin:0`. 라벨·알약·바는 `valign:middle·align:center`.

## 반복 패턴
표지 / 키-밸류 개요 / 타임라인 / 한눈에 리스트 / 보라 디바이더 / 카드 그리드(2×N·3×N) / 정부양식 푸터(부처명·페이지·발표일) / 다크 액션.
모든 슬라이드에 시각요소 1개 이상(아이콘·카드·이미지·통계). 제목 밑 데코라인 ❌.
